
# eEngine
A physics engine writen in [rust](https://www.rust-lang.org/) over piston2d

See the main file for usage examples.

### Status :

**Currently Supported**

*Rigid Body*
```json
Shapes : Rectangle
Laws : Gravity
```

**Planned**

*Rigid Body*
```json
Shapes : Circle
Laws : Collision
```

### Goals out of scope of projects :
1. Camera movement
2. Scaling
